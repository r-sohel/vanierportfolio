Saint-Pierre Le Vignoble Label Design

Assignment

Wine Bottle Design

Tools Used

Illustrator, Indesign & Photoshop

Year

2023

For this project I had to create a series of 2 labels for the brand Saint-Pierre Le Vignoble. The label design is different to the actual real life designs but connected to the brand identity.

Requirements:
Label front side:
Name of the wine
Type of wine (optional)
Alcohol percentage
Year, region and volume.
Label back side:
Wine description
Barcode
